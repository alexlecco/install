# Node with NVM
```bash
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.0/install.sh | bash
$ nvm --version
$ nvm install node
$ nvm use node
$ nvm install --lts
$ nvm use --lts
```

# Firefox Dev Ed with PPA
```bash
sudo add-apt-repository ppa:ubuntu-mozilla-daily/firefox-aurora
sudo apt-get update
sudo apt-get install firefox
```
